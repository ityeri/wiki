# 스폰상점 규칙

## 스폰 상점 규칙

개인 공장이나 건축물 금지\
(공용으로 사용 목적 가능/관리자에게 선 문의)

건축물이나 조경 등 파괴 절대 금지

## 스폰 상점

상점 부지는 정해져 있는 틀에서만 건축 가능합니다\
상점 부지 틀 임의로 생성 불가합니다

건축 예정 상점 부지에 표지판으로 닉네임 표시 필수\
(1주 이상 건축하지 않을 시 표지판은 철거 될 수 있습니다)

2주 이상 운영하지 않을 시 상점 철거 될 수 있습니다

상자 방치, 과하게 적은 판매 물품, 실질적 플레이 시간, 접속 시간\
등을 모두 고려하여 판단 후 임의로 철거 될 수 있습니다

상점 공동 명의 금지 (1인 1상점)

회로 금지(누가봐도 간단한 회로는 가능)

부계정 및 지인 명의 사용 금지\
(상점주와 실제 관리인 명의 대조 할 것임)

현수막 금지

상점 바닥에 구멍 방치 금지

조명작업/반블럭 등으로 몹 스폰 방지

상점 간 연결 금지

발광먹물 상점 당 1개만 사용 가능

겉날개 판매 금지

폭발물 판매 금지\
(ex. tnt, 엔드 수정, 화약, 위더스켈레톤 머리 등)\
(창고든 어디든 상점 내부에서 적발 시 전체 폐기)

위 규칙은 후에 추가, 수정 될 수 있으며 위반 시 관리자 임의로 제제 할 수 있음을 알립니다
